# Sniper Duels - Luau Gun System

A first-person sniper duel gun system for Roblox, replicating the mechanics from SNIPER DUELS.

## Setup Instructions

1. Create a new Roblox Place
2. Insert all scripts into appropriate locations in StarterPlayerScripts, ServerScriptService, etc.
3. Create the GUI elements as described in GuiComponents

## Core Mechanics
- First-person sniper aiming
- Scope zoom (right-click / Right Shift)
- Ballistic raycasting with bullet drop
- Damage falloff over distance
- Reload system (R key)
- Hitmarkers on impact
- Kill feed
- Weapon sway
- Recoil system
